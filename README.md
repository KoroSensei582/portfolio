# portfolio

Personal site for Shahir Ahmed — electrical engineering at Michigan State.

## Tech stack

- [Astro 5](https://astro.build) — static site generator
- [Tailwind CSS 3](https://tailwindcss.com) — utility styling, JIT
- [TypeScript](https://www.typescriptlang.org) — strict mode
- [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) — markdown project writeups
- Self-hosted variable fonts: Bricolage Grotesque (display), JetBrains Mono (body)

## Local development

```bash
git clone https://github.com/<you>/portfolio.git
cd portfolio
npm install
npm run dev
```

Dev server runs at <http://localhost:4321>.

### Scripts

| Command | Action |
| --- | --- |
| `npm run dev` | Start dev server with hot reload |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview the production build locally |
